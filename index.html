Detecção de Notas Falsas em Imagens de Cédulas Bancárias

Situação Problema: Como detectar irregularidades em cédulas que possam caracterizar uma fraude?

Para responder a questão, foi criado um modelo de previsão da legitimidade de notas bancárias é apresentada neste projeto, mostrando a aplicação vantajosa da inteligência artificial na detecção de atividades fraudulentas. Demonstrando uma abordagem inovadora no combate à fraude, este projeto ilustra o potencial de técnicas de machine learning ao utiliza o modelo de árvore de decisão e seu algoritmo CART na detecção de fraudes.

Pacotes destinados ao Machine Learning

Para a devida aplicação do modelo, bibliotecas essenciais Machine Learning como numpy, pandas, sklearn e graphviz, foram cruciais para a realização do projeto.

Dados do Projeto

O projeto faz o download de um conjunto de dados (dataset) a partir de um URL usando comandos do Python. Em seguida, os dados são lidos em um DataFrame do Pandas para análise e processamento. As características incluem variância, assimetria, curtose, entropia e a variável de classe que indica se a nota é falsa ou genuína.
